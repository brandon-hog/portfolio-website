A high-performance, responsive personal portfolio engineered to showcase software development projects, systems engineering experience, and academic background. Built with a modern, type-safe frontend stack to ensure scalability, maintainability, and a seamless user experience.

## Features

* **Dynamic Project Filtering:** A responsive grid interface allowing employers to search and filter technical projects by title, programming language, and category.
* **Centralized Data Management:** Static data (projects, education) is decoupled from UI components using TypeScript interfaces, adhering to the single responsibility principle.
* **Client-Side Routing:** Fast, seamless navigation between the landing page, education history, and detailed project views using React Router.
* **Modern Styling Engine:** Utilizes Tailwind CSS v4's Vite-native integration for zero-config, highly optimized utility-class styling.

## Architecture & Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS v4
* **Routing:** React Router v6

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18+ recommended) installed on your local machine.

### Installation

1. **Clone the repository:**
```
  git clone https://github.com/brandon-hog/portfolio.git
  cd portfolio
```

2. **Install dependencies**:

```
npm install
```

3. **Start the development server**:

```
npm run dev
```

4. **Build for production**:

```
npm run build
```
